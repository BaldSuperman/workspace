#适配知识
-为了适配retina屏幕，可以把2倍的图片进行压缩
- 背景图也可以进行缩放
    - background新属性 bcakground-size
        - length:用长度值决定指定背景图像大小，不允许负值、
        - percentage 用百分比指定背景图像大小，不允许负值
        - auto 背景图像的真实大小
        - cover：将背景图像等比缩放到完全覆盖容器，背景图像可能超出容器
        - contain 将背景图像等比缩放到宽度火高度于容器的宽度和高度相等，背景图像始终被包含在容器内

    - <aside> 侧边栏
    - <footer> 页面底部 页脚
